DP_epidemiology documentation
=============================

**DP_epidemiology** (/lu'make/) is a Python library for differential private release of financial data in the context of epidemiology. It is built on top of the `OpenDP <https://opendp.org/>`_ library
and offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
